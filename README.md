# go-practices
---
## 项目介绍
个人GO练习项目

#### base: 基础语法以及基础库练习


#### some_practices: 一些练习项目
- demochain: 区块链demo
- tree: 二叉树
- web_crawl: 网络爬虫(异步练习)
- gossip: 分布式数据库系统中各个副本节点同步数据协议

#### third_packages: 第三方库的使用练习
- kingpin: 命令行解析库(类似于go自己的解析库flag)
- protocol_buffers: googele 的序列化数据结构的协议
- testify: 测试库
